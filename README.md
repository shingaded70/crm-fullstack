# Full Stack CRM System

## Tech Stack
- Spring Boot (Backend)
- React.js (Frontend)
- JWT Authentication
- H2 Database
- Chart.js Analytics

## Features
- User Authentication (JWT)
- Role-based Access (Admin / User)
- Customer Management (CRUD)
- Lead Management
- Task Management
- Sales Tracking
- Dashboard Analytics (Bar + Pie Charts)

## Setup Instructions

### Backend
cd crmbackend
mvn spring-boot:run

### Frontend
cd crmfrontend
npm install
npm start

Backend runs on port 8080  
Frontend runs on port 3000
